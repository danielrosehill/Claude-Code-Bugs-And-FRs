### Problem Statement

I find it really distracting to have the warning showing all the time even when I'm only at 75% (and in any event, I have a work account that I can also use!).

### Proposed Solution

A couple of ideas for other folks like me who ..... (struggling for words) ... find UI elements like this stress-inducing and distracting:

1) The warning only shows twice. Once at 70% of usage, once at 95% of usage (your quota is almost reached). Users get the message but don't have it in their face all the time. Or just send notification emails (or do both).

2) Make this a setting. Those blessed with normal brains can see this all the time. Folks like me can choose to either eliminate or tone down the usage messages.

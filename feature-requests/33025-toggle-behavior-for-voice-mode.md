### Problem Statement

Voice mode looks very promising. However, the push-to-talk functionality is cumbersome for using it to dictate longer instructions and prompts.

### Proposed Solution

It would be great if users were able to choose between the hold-to-talk or push-to-talk functionality and a press-to-start-and-stop approach using the spacebar. 

The first might be favored for those who like to send in short messages, and the second for more expansive prompts which might take a couple of minutes to dictate, in which case holding down the spacebar for the entire duration becomes quite cumbersome.

# Claude Code — Bugs & Feature Requests

Staging area for Claude Code issues before filing upstream at [anthropics/claude-code](https://github.com/anthropics/claude-code/issues).

## Layout

```
bugs/             # Bug reports
feature-requests/ # Feature requests
```

Each item is a single Markdown file. Once filed upstream, the file gets a header link to the upstream issue.

## Status legend

- **draft** — being written
- **needs-repro** — needs a clean reproduction before filing
- **ready** — ready to file upstream
- **filed** — filed upstream (link in header)
- **closed** — resolved upstream
